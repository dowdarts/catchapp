### Playing the Game
1. Enter your name or play as guest
2. Each round shows a target out (61-100)
3. Record how many darts it took to finish
4. Complete all 40 outs for a maximum of 120 points

### Keyboard Shortcuts
- Press `0` for miss (7+ darts)
- Press `2` for 2 darts
- Press `3` for 3 darts
- Press `4`, `5`, or `6` for 4-6 darts
- Press `Backspace` to undo
